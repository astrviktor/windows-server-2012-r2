На HIPER-V будет крутится несколько экземпляров Windows Server 2012 R2, поэтому лучше сделать базовый образ и от него дифференциальные жесткие диски.

Для клиентских OS аналогично.

Начиная с windows 8 HIPER-V идет в комплекте, надо просто добавить.
В windows 10 добавление: "Параметры" / "Приложения" / Внизу списка пункт "Программы и компоненты" / "Включение или отключение компонентов Windows" / "HIPER-V"

01. Создаем виртуальный жесткий диск:
	VHDX
	Динамический размер
	Имя ws2012r2-base.vhdx
	Указать расположение
	Указать размер 50GB

02. Создаем виртуальную машину
	Имя: ws2012r2-base
	Поколение 1
	Память при запуске: 1024
	Подключение: Default Switch
	Жесткий диск: Подключить имеющийся ws2012r2-base.vhdx
	После создания в dvd-дисковод подключить образ windows server 2012 r2

03. Запускаем машину, устанавливаем windows
	до изменения групповых политик паролей ставим пароль qwerty`123
	после загрузки нужно подготовить windows для включения как в первый раз для получения правильных SID (Security ID) и RID (Relative ID). Чтобы это сделать, нужно:
		Запустить cmd от имени администратора
		перейти по пути cd c:\windows\system32\sysprep
		запустить sysprep.exe
		выбрать опции ...(OOBE), Generalize и Shutdown
		удалить все контрольные точки чтобы получить финальный вид базового виртуального диска
		на всякий случай поставить на виртуальный диск атрибут "только для чтения"
