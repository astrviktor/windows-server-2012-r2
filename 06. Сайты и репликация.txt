Настроим два сайта, Main и Far, как отобраение двух физически разнесенных офисов

Нужно:
	Создать 2 сайта
	Разнести dc по сайтам
	Настроить диапазоны IP для сайтов
	Настроить транспорт и параметры репликации


01. Заходим в оснастку Active Directory Site and Services
	Переименуем Default-First-Site-Name в Main
	Sites / New site Добавим новый сайт Far
	Перетаскиваем сервер dc3 в сайт Far


Транспорт репликации между сайтами:
RPC - синхронный (посылает запрос одному dc, ждет ответа, другим ничего не отправляет)
SMTP - асинхронный (может одновременно запросить несколько dc и постепенно получать с них информацию)

Приоритет маршрута репликации между сайтами:
- Маршрут с меньшей стоимостью
- Если стоимость одинаковая, то с меньшим количеством прыжков
- Если стоимость и количество прыжков одинаковое, то по алфавиту названий сайтов


02. Настраиваем SiteLink
	Inter-Site Transports / IP / ... <- RPC over IP
	Переименуем Default в Main-Far site link
	Добавляем SiteLinks и попарно указываем сайты, Cost и периодичность репликации
	Можно настраивать расписание
	Финальный маршрут репликации будет строиться по минимальной стоимости
	Поставим cost 10 и time 15 минут

Репликация по умолчанию между dc внутри сайта: 15сек + 3сек
Репликация по умолчанию между dc в разных сайтах: 3часа + 15мин

03. Создаем подсети для разных сайтов
	Subnets / New Subnet 
	192.168.11.0/24 для Main
	192.168.22.0/24 для Far

Глобальный каталог - чтобы знать краткую информацию о всех обьектах всех доменов
Servers / NTDS settings свойства - [x] Global Catalog
Желательно чтобы все dc были также серверами Global Catalog. Если нет, может возникнуть конфликкт с ролью FSMO


04. Проверка репликации
	Добавим пользователя test на dc1 сайта Main:
		остнастка Active Directory Users and Computers / Users / New User / test (qwerty`123) 
	Проверим, на dc2 test появился почти мгновенно (за время открытия остнастки)
	На dc3 test должен появиться через 15 минут, которые мы настроили (+- 15 минут).
	Подождем 5 минут, и форсируем репликацию:
		repadmin /syncall DC3 dc=ecorp,dc=local
		или
		Servers / NTDS Settings / Replicate from/to


Посмотреть состояние репликации: repadmin /showrepl

Для проверки на какой AD сайт мы вошли, на клиенте надо выполнить команду:
	nltest /server:client.ecorp.local /dsgetsite
